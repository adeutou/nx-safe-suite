# @nx-safe-suite/server-cache

> Multi-tier (memory -> Redis -> source) server-side cache for Next.js with tag-based invalidation.

## Status

🚧 Work in progress: part of the [next-safe-suite](../../README.md) monorepo. API not yet stable.

## Installation

```bash
pnpm add @nx-safe-suite/server-cache
```

## Usage

```ts
import "@nx-safe-suite/server-cache";
```
